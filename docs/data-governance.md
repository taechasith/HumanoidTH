# Data Governance

Store public metadata, source URLs, short excerpts, and derived labels. Do not redistribute full text unless a source license permits it.

Owned inventory defaults to private. Public-safe views hide serial numbers, exact storage location, purchase documents, and security-sensitive repair notes unless marked public-safe.

Contribution claims remain pending until reviewed. Contributor profiles must not expose private contact details.

Corrections and takedown requests should be submitted through `/submit-data` as `correction_request`.
